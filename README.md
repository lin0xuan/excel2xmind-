# excel2xmind-兼容存在合并单元格形式

##excel示例
#
|一级节点1|二级节点1|三级节点1|...
#
|一级节点1|二级节点2|三级节点2|...
#
|一级节点2|二级节点2|三级节点2|...
#
|一级节点2|二级节点2|三级节点3|...
...

不需要有表头，直接放入节点的层级关系即可，生成的xmind需要微调一下，而且打开的时候会显示是旧版本生成的xmind，因为新版本的xmind似乎会打不开，欢迎大家迭代更新，开拓新用法。

#excel中存在合并单元格的一级节点、二级节点等等都是可以正确识别的。
#另外还有xmind转excel的代码附上，有需要的同学可以自取。
